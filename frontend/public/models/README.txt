Place your heart.glb 3D cardiac model file here.

If this file is not present, the application will automatically use a procedural 
3D heart model built from Three.js geometries (sphere lobes + aorta + animation).

Recommended sources for heart.glb:
- Sketchfab: search "heart anatomy" (filter: free, glb)
- Poly.pizza
- Google Poly archive

After placing the file, restart the backend server.
